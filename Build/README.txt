JMDM - 실시간 데미지 미터 
======================== 
 
🚀 단일 실행파일 버전 
 
📋 필수 요구사항: 
- Windows 10/11 (64비트) 
- Npcap 드라이버 설치 필수! 
  다운로드: https://npcap.com/ 
- 관리자 권한 
 
🎮 사용 방법: 
1. Npcap 설치 
2. JMDM.bat를 관리자 권한으로 실행 
   또는 JMDM-Start.vbs 더블클릭 
3. 네트워크 인터페이스 선택 
4. 웹브라우저에서 http://localhost:8080 접속 
 
📁 파일 구성: 
- JMDM.exe (모든 기능 포함된 단일 실행파일) 
- JMDM.bat (관리자 권한 실행 스크립트) 
- JMDM-Start.vbs (더블클릭 실행용) 
- content/ (웹 인터페이스) 
 
⌨️ 단축키: 
- C: 네트워크 인터페이스 변경 
- Q: 프로그램 종료 
 
⚠️ 주의사항: 
- 방화벽에서 차단 시 허용 필요 
- 안티바이러스에서 오탐 시 예외 처리
